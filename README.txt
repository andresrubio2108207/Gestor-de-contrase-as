

Descripción:


Este proyecto representa una gestión de usuarios en Python que combina una interfaz gráfica
con Tkinter y una terminal interactiva con Rich la cual permite crear usuarios,
generar contraseñas aleatorias, validar credenciales y mostrar los datos
en una tabla visual todo almacenado en archivos locales.

---REQUERIMENTS---

LIBRERIAS:

Rich: pip install rich
Pillow: pip install Pillow

Es posible que se necesite la version mas reciente de pip

(PARA WINDOWS)

Pip install: python -m ensurepip --upgrade
Pip last version: python -m pip install --upgrade pip 

---PASOS A SEGUIR PARA UTILIZARLO---

1. Instala los requerimientos descritos anteriormente
2. Crea tu usuario en el main.py
3. Inicia el archivo launcher para activar la interfaz e ingresa tu usuario y contraseña descrito anteriormente en el main.
